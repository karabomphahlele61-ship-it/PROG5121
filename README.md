QuickChat Console Application
Project Overview

This project is a console-based chat application developed in Java for the Programming PoE. The purpose of the application is to allow users to register, log in and send messages through a simple QuickChat system.

The project was developed using Object-Oriented Programming principles and includes unit testing with JUnit as well as version control using GitHub.

Features Implemented

**Part 1 — Registration and Login**
User registration
Username validation
Password complexity validation
South African cellphone number validation
User login authentication
Login status messages

**Part 2 — Messaging System**
Send messages
Message ID generation
Message hash generation
Recipient validation
Message length validation
Store messages in JSON file
Total messages counter
Menu-driven console application

**How to Run the Program**

Open the project in NetBeans.
Run the Registration.java file.
Register a new user.
Login using the registered credentials.
Use the QuickChat menu to send messages.
